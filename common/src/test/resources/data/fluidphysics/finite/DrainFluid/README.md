# 测试：流体物理-有限模式-原版-抽取流体

## 测试描述

该测试用于验证流体抽取算法的正确性。

该测试模拟了一个一次性最多抽取 1 B 的泵直接用天圆地方的流体抽取算法抽取流体时，算法是否能够正确进行抽取。

## 样例格式

输入格式为一个描述结构的格文件。在格文件的元数据扩展（ext）处，需填写以下字段：

- expected：一个 Map。
    - unit：可填写 mB 或 QB，为 drained 的单位。
    - drained：期望抽取到的流体量。
- drain_at：一个三元组，即抽取流体的坐标。

## 样例 #1

### 输入 #1

```
---
name: A
ver: 1
upon: liquids
import: 天圆地方:soil
default: 土0
ext:
  drain_at: [2,2,2]
  expected:
    unit: mB
    drained: 1000
---
略

石石石
石㵘石
石石石

略

```

## 说明/提示

### 样例 1 解释：

抽取地点为第二层第二行第二列，为㵘（满格水），因此期望抽取 1000 mB。
