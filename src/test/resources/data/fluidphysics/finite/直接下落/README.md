# 测试：流体物理-有限模式-原版-直接下落测试

## 测试描述

本目录中的测试文件用于验证有限模式下原版流体在“直接下落”流动规则下的行为。

每个测试用例由两个文件组成：

```
<name>.in.格   初始场景
<name>.ans.格  进行一次直接下落后的期望场景
```

测试程序会读取 `.in.格` 文件构建初始世界，并执行且仅执行**一次**流体直接下落更新，然后将结果与 `.ans.格` 文件中的结构进行比较。

## 输入格式

输入格式为一个描述结构的格文件。在格文件的元数据扩展（ext）处，需填写以下字段：

- fall_pos：一个三元整数组，表示需要下落的流体所处的坐标位置。

## 答案格式

答案格式为一个描述结构的格文件。无特别要求的扩展字段。

## 输入输出样例 #1

### 输入 #1

```
---
ver: 1
ext:
  fall_pos: [2,2,2]
---
石石石
石灬石
石石石

石石石
石灬石
石石石
```

### 答案 #1

```
---
ver: 1
---
石石石
石溢石
石石石

石石石
石〇石
石石石
```

## 说明/提示

### 样例 1 解释：

输入定义了位于 (1,1,1) 的流体开始下落，(1,1,1) 的流体是灬，也就是四层水，四层水下落和下方的灬合成八层**流动水**，即**溢**。
