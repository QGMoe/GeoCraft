# 测试：流体物理-有限模式-原版-单层坡度流动

## 测试描述

该测试用于验证单层坡度流动算法的正确性。

单层坡度流动算法是一个计算期望流动方向的算法。其应该具有以下性质：

- 在平地上，无流动方向；
- 在崎岖的地面上，单层坡度流动算法应选择单层坡度流动距离范围内坡度最大的方向。

单层坡度流动算法的单层坡度距离决定了搜索范围。
例如，当其为 2 时表示应当搜索下面这个范围的区域。
其中**丶**表示单层静态水，**中文数字**表示搜寻到当前位置时的坡度搜寻距离，
**边**表示最外层会被探测的位置，**〇**表示不会被探测的位置：

```
〇〇〇边〇〇〇
〇〇边二边〇〇
〇边二一二边〇
边二一丶一二边
〇边二一二边〇
〇〇边二边〇〇
〇〇〇边〇〇〇
```

## 输入格式

第一行包含三个整数 $H$、$R$、$C$，分别表示层数、每层行数和每层列数。

第二行包含三个整数 $X$、$Y$、$Z$，表示需要触发单层坡度流动的位置，
对应文本中第 $Y+1$ 层、$Z+1$ 行、$X+1$ 列的方块，必须是单层水，无论静态动态或处于垂直下落状态。

接下来 $H \times (R+1) -1$ 行包含 $H$ 个之间用**单个空行**间隔的 ，
内部固定为连续 $R$ 行，每行共 $C$ 个方块的用字符序列表示的平面。
每个平面所属的行的末尾可以附加上 **#内容** 以进行注释。

再之后的行不会被读取，可以填写任意内容。

## 答案格式

答案包含了单层坡度流动算法应当得出的方向集合，每行表示一个方向，大小写不敏感。可用的方向如下：

| 行内容（顿号表示有多种选择）          | 方向 | 
|-------------------------|----|
| EAST、RIGHT、东、右          | 正东 |
| WEST、LEFT、西、左           | 正西 |
| SOUTH、BACK、BACKWARD、南、后 | 正南 |
| NORTH、AHEAD、HEAD、北、前    | 正北 |

技巧：文本输入中，可以直接通过上北下南左西右东判断方位。同时，如果用汉字则可以在一行内连着写而不用换行。

## 输入输出样例 #1

### 输入 #1

```
3 3 3
1 1 1
〇〇〇
〇石〇
〇〇〇

〇〇〇
〇丶〇
〇〇〇

岗闪岗
闪岗闪
岗闪岗
```

### 答案 #1

```
东西南北
```

## 说明/提示

#### 样例 1 解释：

单层水的前后左右世界对称分布，且都可以流下，流动难度等大，因此返回东西南北四个方向。

---

#### 约定

对于世界外围的方块，默认均为黑曜石。

水的单层坡度搜寻距离固定为 4，岩浆则为 2。

要求 $0 \leq H,R,C \leq 2147483647$，$0 \leq X \le C$，$0 \leq Y \le H$，$0 \leq Z \le R$ 。
