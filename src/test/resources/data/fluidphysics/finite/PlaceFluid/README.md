# 测试：流体物理-有限模式-原版-放置流体

## 测试描述

该测试用于验证流体放置算法的正确性。

该测试模拟了一个玩家将指定水/岩浆方块放置到世界上时，算法是否能够正确寻找到可填充流体的空间并判断是否能够放置。

## 样例格式

输入格式为一个描述结构的格文件。在格文件的元数据扩展（ext）处，需填写以下字段：

- place：一个 Map。
  - at：一个三元整数组，表示需要放置流体的坐标位置。
  - fluid：需要放置的流体方块词块。
- expected：一个布尔值，期望是否能够放置。

### 样例 1 解释：

第 2 层均为单层水，拥有充足的空间放置**㵘**，也就是满格水（Quanta = 8, Level = 0）方块。
